# Quizz Nico — Révision HLT Conducteur de train SNCB

Quiz interactif de révision basé sur le **Livret HLT Fascicule I** (Conducteur de train SNCB), chapitres 1 à 3, Avis 01/02/03 B-TO 2026.

## Contenu
- **105 questions** réparties en 3 niveaux : 35 débutant, 35 moyen, 35 difficile (QCM et QRM).
- Explications détaillées pour la bonne **et** la mauvaise réponse.
- Couverture des 3 chapitres : Structure des chemins de fer belges, Connaissances du conducteur, De la prise de service à la fin de service.

## Fonctionnalités
- Feedback immédiat (mode entraînement) ou **mode examen chronométré**.
- **Score et progression** sauvegardés localement (reprise des questions ratées).
- Filtrage par **chapitre** et par **niveau**.
- **Fiches de synthèse / mémo** par chapitre.

## Utilisation
Ouvrir `index.html` dans un navigateur (le fichier `data.js` doit être dans le même dossier).
Aucune connexion internet requise.

## Hébergement (réviser depuis le téléphone)
Active GitHub Pages : Settings → Pages → Source = branche `main` / dossier `/root`.
Le quiz sera accessible via une URL du type `https://boutiqueambiancejapon-sketch.github.io/quizz-nico/`.
